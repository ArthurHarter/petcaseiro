# 🐾 PetCaseiro

> Blog de cuidados, dicas e produtos para cães e gatos — hospedado via GitHub Pages.

**URL do site:** `https://SEU-USUARIO.github.io/petcaseiro/`

---

## 📋 Sobre o site

O **PetCaseiro** é um blog de conteúdo sobre cuidados com pets voltado para tutores brasileiros. O site combina artigos informativos com recomendações de produtos afiliados do Mercado Livre, além de espaços para anúncios via Google AdSense.

**Objetivo:** Gerar tráfego orgânico (SEO) e monetizar por meio de:
- 🛒 Links de afiliado do Mercado Livre
- 📢 Anúncios Google AdSense
- 📬 Captura de e-mails (newsletter)

---

## 📁 Estrutura de arquivos

```
petcaseiro/
├── index.html        ← Página inicial (blog + artigos)
├── produtos.html     ← Comparativos e produtos afiliados
├── caes.html         ← (em breve) Artigos sobre cães
├── gatos.html        ← (em breve) Artigos sobre gatos
└── README.md         ← Este arquivo
```

---

## 🚀 Como usar

1. Substitua todos os links `https://www.mercadolivre.com.br/` pelos seus **links de afiliado reais** do ML
2. Substitua `ca-pub-XXXXXXXXXXXXXXXX` pelo seu **ID do Google AdSense**
3. Suba os arquivos no GitHub e ative o **GitHub Pages**
